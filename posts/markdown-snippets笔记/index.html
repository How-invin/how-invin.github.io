简介

使用vim的vim-snippets插件，熟悉vim的markdown的默认代码片段。

正文

网址 图片 评论

snippet [
	[${1:text}](http://${2:address})

按下[之后出发超链接的代码片段

snippet ![
	![${1:alttext}](${2:/images/image.jpg})

按下![触发图片的代码片段

snippet <
	<http://${1:url}>

按下<之后触发链接的代码片

snippet /* "Comment"
<!-- ${1:${VISUAL}} -->$0
endsnippet

按下/*之后触发链接的代码片

加粗 斜体 加粗斜体 代码片段

snippet **
	**${1:bold}**

按下*触发斜体的代码片段

按下**触发加粗的代码片段

按下***触发斜体加粗的代码片段

snippet ```
	\`\`\`${1}
	${0:${VISUAL}}
	\`\`\`

按下```触发代码块的代码片段

建立表格

post_jump "create_table(snip)"
snippet "tb([1-9][1-9])" "Fancy table" br
`!p snip.rv = match.group(1)`
endsnippet

tb 后面加数字x，数字y出发制表的代码片段
